# ADR-18: Local Database Schema
**Status**: APPROVED  
**Author**: Antigravity Core Architect  

---

## 🎯 Purpose
Saves session settings, workspace histories, and persistent environment configurations locally.

## 🗄️ Stores
* **Settings**: Editor configurations, theme modes, proxy rules.
* **Workspace History**: Previously opened workspace directories.
* **Tabs State**: Monaco Editor tab configurations.
